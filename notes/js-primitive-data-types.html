JavaSript Primitive Data Types

JavaScript contains 6 primitive data types. These data type are immutable, meaning they cannot be changed. An object is not a primitive data type.

Primitive Data Types:

  • String
  • Symbol
  • Number
  • Boolean
  • Undefined
  • Null

String

Strings are a collection of characters within quote marks.

Example:

'this is a string'

Properties:

  • .length - returns: number - number of characters, including whitespace, in string

Methods:

  • .toUpper() - input: none; returns: string - makes the string characters UPPERCASE
  • .toLower() - input: none; returns: sting - makes the string characters lowercase
  • .charAt() - input: number; returns: string - the character or characters located at the index
  • .indexOf() - input: string; returns: number - the index of the string, -1 means that string not found; case-sensitive
  • .lastIndexOf() - input: string; returns: number - the index of the last occurrence of character(s); case-sensitive
  • .includes() - input: string; returns: boolean - true means string was found, false means string not found; case-sensitive
  • .startsWith() - input: string; returns: boolean - true if string starts with character or characters, false if not; case sensitive
  • .endsWith() - input string; returns: boolean - true if string ends with character or characters, false if not; case-sensitive
  • concat() - input: none; returns: string - combines two or more strings; using the + operator is a shortcut
  • trim() - input: none; returns: string - removes any whitespace from string
  • repeat() - input: number; returns: string - repeats string the stated number of times (input) resulting is a concatenated string

Functions:

  • String() - input: number; returns: string - changes number data type to string data type

Escape Characters:

  • \' = Single quote marks
  • \" = Double quote marks
  • \n = End of line
  • \r = Carrage return
  • \t = Tab
  • \\ = Backslash

Template Literals:

Special type of string declared using the backtick character (`). This form of string is considered a super-powered string. It behaves like a normal string, but it has the extra power of string interpolation. It is not uncommon to see code with all strings created as template literals.

  • Able to use both type sof quote marks without using escape characters
  • Allows interpolation of JavaScript code
  • \n = End of line
  • \r = Carrage return
  • \t = Tab
  • \\ = Backslash
