localStorage

localStorage is a javascript object that stores data with no experation date. Data will not be deleted even if the browser is closed. Data is stored in key/value pairs on the user's computer. As an HTML5 specification, localStorage is supported by major browsers - including IE8. Limitations include no data protection, data can be accessed by any code on the web page, limited to 5MB across all major browsers, data is only stored on the browser (not a substitue for a server based database), and localStorage is synchronous. Sensitive user information should never be stored in localStorage.

Methods include:

  • setItem(): Add key and value
  • getItem(): Retrieve item from localStorage by key
  • removeItem(): Delete item from localStorage by key
  • clear(): Clear all items from loacalStorage
  • key(): Passed a number to retrieve the key of a localStorage
