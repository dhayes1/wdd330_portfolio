Comments

Comments are a tool that can be used by programmers to make their code more readable by explaining the intent of the code. Comments can also be utilized to prevent a single line of code from executing or used to prevent blocks of code from execution. Comments are ignored during code execution.

Javascript Comments:

  • Single Line comments start with //. The program will ignore any text that begins with // until the next line.

    • Example:

    // this is a single line comment

  • Multi-line comment start with /* and ends with */. The program will ignore any text contained within /* and /*

    • Example:

    /* this is a multi line comment
    this is a multi line comment
    this is a multi line comment
    this is a multi line comment */
