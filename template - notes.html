Javascript Comments

Comments are a tool that can be used by programmers to make their code more readable by explaining the intent of the code. Comments can also be utilized the prevent code execution. Comments are ignored during code execution.

Types:

  • Single Line - used to comment a single line of code
    • Symbol: //
  • Multi-line - used to comment a block of text
